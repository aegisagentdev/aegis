# Glossary

| Term | Definition |
|------|-----------|
| **AMM** | Automated Market Maker — a DEX protocol that uses liquidity pools instead of order books |
| **Arbitrum Orbit** | A framework for launching customized L2/L3 chains on top of Ethereum via Arbitrum |
| **Bytecode** | The compiled contract code deployed on-chain; what `eth_getCode` returns |
| **DANGER** | Highest severity level; indicates a blocking issue that forces a NO verdict |
| **DEX** | Decentralized Exchange — a trading venue that operates on-chain without a central operator |
| **EIP-1967** | Standard for proxy storage slots; used to detect upgradeable contracts |
| **EOA** | Externally Owned Account — a regular wallet address with no contract code |
| **ERC-20** | Standard interface for fungible tokens on Ethereum and compatible chains |
| **eth_call** | JSON-RPC method that simulates a transaction without executing it (read-only) |
| **FCFS** | First-Come-First-Served sequencing — transactions ordered by arrival, not gas price |
| **Honeypot** | A token that allows buying but blocks selling, trapping the buyer's funds |
| **L2** | Layer 2 — a scaling solution that processes transactions off the main Ethereum chain |
| **MEV** | Maximal Extractable Value — profit extracted by reordering, inserting, or censoring transactions |
| **Proxy** | A contract pattern where the logic can be upgraded by changing the implementation address |
| **RPC** | Remote Procedure Call — the JSON-RPC interface used to communicate with blockchain nodes |
| **Rug pull** | When a project's creators drain the liquidity pool, crashing the token price to zero |
| **Slippage** | The difference between expected and actual trade price, caused by price movement during execution |
| **Stock token** | A tokenized debt instrument on Robinhood Chain that tracks an equity price |
| **TVL** | Total Value Locked — the total amount of assets deposited in a DeFi protocol |
| **Uniswap V3** | A concentrated-liquidity AMM where LPs provide liquidity within specific price ranges |
| **Verdict** | Aegis's final assessment: GO, CAUTION, NO, or UNKNOWN |
