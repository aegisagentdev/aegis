# Getting Help

## Before Opening an Issue

1. **Check the [README](README.md)** — most configuration and usage questions are answered there.
2. **Run `hoodtrade doctor`** — this validates your RPC connection and reports the chain id.
3. **Check [CHANGELOG](CHANGELOG.md)** — your issue might be fixed in a newer version.

## Reporting Bugs

Open an issue with:

- Hood Trade version (`hoodtrade version`)
- Python version (`python --version`)
- The full command you ran
- The full error output
- Your OS (macOS / Linux / Windows)

## Feature Requests

Open an issue with the `enhancement` label. Describe:

- What you want to do
- Why the current behavior doesn't cover it
- How you'd expect it to work

## Security Issues

See [SECURITY.md](SECURITY.md). Do **not** open a public issue for security vulnerabilities.

## Questions

For general questions about DeFi safety, Robinhood Chain, or how to interpret a verdict,
open a discussion or issue tagged `question`.
